Deutsch-Daily is a vibe coding experiment using Claude to create the whole thing as a  Progressive Web App (PWA).
The initial app was done on my iPhone in around 90 minutes using Claude and Githup to serve the pages in an airline loung in MSP (Minnesota) while waiting for my flight.
Each lesson consists of two nouns, a verb, an adjective and two sentences to illustrate those items.
Each Common European Framework of Reference for Languages (CEFR) level, A1, A2 etc. is presented as a set of 25 lessons grouped into blocks of five. 
After each block of 5 lessons there is a short test consisting of twelve questions on the contents of the 5 lessons in that block. 
Pass rate is 90% so a you have to get 11 or qw correct to pass.
Af the last test of the level is completed, there is a giant test of 40 questions on all the lessons in the level.
Again, pass rate is 90%.
After that you can procedd to the next CEFR level.
